# 哈哈镜
### ***给图片添加滤镜和贴纸以及实时滤镜，支持14种滤镜效果***


### 一、介绍
##### 这是一个关于在图片上添加贴纸以及实时滤镜的项目，动图里演示了操作方法。

>+ 项目里要求，要实现下面以下功能：
>+ 实时滤镜；
>+ 对图片添加滤镜；
>+ 对图片添加标签；
>+ 对图片添加贴纸。
